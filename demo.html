BoilerSASS

This is the documentation and an example of BoilerSASS usage.
You can download the code from GitHub https://github.com/Gix075/boilerSass

BoilerSASS is a simple boilerplate for start new SASS projects.
BoilerSASS is based on HTML5 Boilerplate and include Grunt as tasks manager.

Vendors Libs

By default it not include anything else but you can install it loading Bootstrap or some icon libraries.
BoilerSASS is also modular, so you can easy remove some partials files or add your owns.

  • Bootstrap 4
  • Normalize.css
  • Font Awesome
  • Material Icons
  • Elegant Icons

You can install vendors libraries using Grunt before starting project. You can select the libraries that you want, simply editing the config.json file.

Patials Files

BoilerSASS is modular, so you can easy remove some partials files or add your owns.
By default it's composed by a mainfile that includes all partial scss files and the external scss libs

How To Install

In order to install BoilerSASS, download the package and follow this steps:

  • Install NPM components
    Open you system console and access BoilerSASS directory on your desktop
    the type $ npm install
  • Configure package
    Open the file config.json and set as true the libraries that you want include on your sass project.
  • Install vendors libs
    Open you system console and access BoilerSASS directory on your desktop
    the type $ Grunt install
  • Using SASS
    Once BoilerSASS is installed you can start coding Sass
    Type $ Grunt on your console to first BoilerSASS compile and to start the watch task that autocomile SASS files when changed.

Now you are ready!
