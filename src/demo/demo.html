boilerSASSSimple boilerplate for SASS/SCSS projects

boilerSASS is very simple and easy to use.
You can integrate it in any project and with other libraries or frameworks like Bootstrap, jQueryUI and so on.

Install and Use

In order to install and use boilerSASS simply

  1. download it from GitHub
  2. enter in dist, get the sass directory and import it in your project as you need
  3. customize the Gruntfile.js if you want to use Grunt or setup you task processor
  4. open sass/_variables.scss and customize values as you need
  5. open sass/_mixins.scss and customize mixins as you need
  6. open sass/boilerSass.scss and customize it as you need, you can remove the @import that you don't need and add your own partials.

Framework

boilerSass provide also a little CSS framework with grid, buttons, typography and so on.
You can simply remove the style you don't nedd removing the partials import in boilerSass.scss.
